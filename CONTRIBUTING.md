# Contributing

Thanks for considering a contribution.

## Quick Start

1. Fork the repository.
2. Create a feature branch.
3. Make your changes.
4. Run local checks:

   ```bash
   node --check script.js
   python3 -m http.server 4173
   ```

5. Open a pull request with a clear summary.

## Style Guidelines

- Keep HTML semantic and accessible.
- Prefer readable, multi-line CSS/JS formatting.
- Add or update `data/projects.json` instead of hardcoding project cards in HTML.
- Keep the site responsive and keyboard-friendly.
